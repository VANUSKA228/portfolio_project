# CHANGELOG

## [0.2.0] — 25.05.2026

### Добавлено

**Панель администратора:**
- Страница управления пользователями с таблицей всех зарегистрированных аккаунтов
- Поиск пользователей по имени прямо в админ-панели (фильтрация на лету)
- Кнопка «Удалить» для удаления отдельных пользователей с подтверждением
- Кнопка «Зайти» для входа в аккаунт пользователя (режим impersonation)
- Массовое удаление пользователей через чекбоксы
- Кнопка «Изменить» — редактирование профиля пользователя (имя, email, роль)
- Пометка «(вы)» возле имени самого администратора в таблице
- Защита от удаления самого себя

**Режим технической поддержки (impersonation):**
- Администратор может временно войти в аккаунт любого пользователя
- После входа отображается плашка «Вы находитесь в режиме технической поддержки»
- Кнопка «Выйти в админ-панель» в профиле пользователя
- В навбаре кнопка «Выйти» заменяется на «Выйти в админ-панель»
- При возврате сессия администратора полностью восстанавливается

**Публичный профиль:**
- Страница пользователя доступна по ссылке `/user/username/` для всех посетителей
- Администратор имеет ссылку «Моя страница» в навбаре

**Поиск пользователей:**
- Живой поиск в навигационной панели (доступен всем)
- Выпадающие результаты с именем и ролью

**Логирование:**
- Все действия администратора записываются в файл `admin_actions.log`

### Исправлено
- Ошибка `NoReverseMatch` для маршрута `logout`
- Конфликт URL с встроенной админкой Django
- Ошибка `IntegrityError` при регистрации
- Ошибка `KeyError` при выходе из режима impersonation
- Дублирование функций в `core/views.py`
