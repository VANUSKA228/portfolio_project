# CHANGELOG

## [0.1.0] — 24.05.2026

### Добавлено
- Инициализация Django-проекта (`portfolio_project`)
- Создание приложений `accounts` и `core`
- Кастомная модель пользователя (`User`) с полем `role` (user/admin)
- Регистрация пользователей (логин + пароль)
- Вход в систему (стандартный LoginView)
- Выход из системы (кастомный logout_view с редиректом на главную)
- Раздельные страницы профиля в зависимости от роли:
  - `user_profile.html` — для обычного пользователя
  - `admin_profile.html` — для администратора (заглушка таблицы)
- Главная страница-заглушка (`home.html`)
- Базовый шаблон с Bootstrap 5 (`base.html`)
- Файлы `.gitignore` и `requirements.txt`
- Документация: `README.md`, `CHANGELOG.md`
